# Пример блога на php (Архитектура MVC)
Реализована регистрация, авторизация, вывод списка пользователей, добавление/редактирование/просмотр статей.

Стек технологий:
* Apache
* PHP
* MySQL
* HTML5/CSS3 (Framework Bootstrap)
* JavaScript
